# 폼생AI Scout

숏폼 제휴마케팅용 리서치/벤치마킹 AI입니다.

## 핵심 기능
- 키워드 입력
- YouTube Shorts 후보 자동 수집
- 틱톡/인스타/쇼츠 URL 수동 입력 분석
- 벤치마킹 리포트
- 조회수 잘 나오는 공통 패턴 분석
- 후킹 패턴 20개
- 자막 패턴
- 촬영구도 패턴
- 15초/25초 숏폼 대본
- 캡컷용 자막 스크립트
- 썸네일 문구
- 다음 영상 아이디어

## 필요한 키
Vercel 환경변수 또는 로컬 `.env.local` 파일에 아래 값을 넣으세요.

OPENAI_API_KEY=본인 OpenAI API 키
YOUTUBE_API_KEY=본인 YouTube Data API 키

## 로컬 실행
npm install
npm run dev

브라우저에서 http://localhost:3000 접속
